Cerebral palsy (CP)
  • Cerebral palsy (CP) is a group of permanent movement and posture disorders that result from non-progressive disturbances that occurred in the developing fetal or infant brain. It affects muscle control, coordination, and balance. The severity and symptoms of cerebral palsy vary widely among individuals. Here's an overview of cerebral palsy
Types of Cerebral Palsy:

1.Spastic Cerebral Palsy:

2.Dyskinetic Cerebral Palsy:
  • Description :: Involves uncontrollable, involuntary movements.

3. Ataxic Cerebral Palsy:
  • Description :: Affects balance and coordination.
  • Subtypes:Shaky movements, difficulty with precision tasks.
4. Mixed Cerebral Palsy:
  • Description :: Combination of spasticity, dyskinesia, and/or ataxia.
